# MIRICO - Редизайн сайта

## О проекте

Современный веб-сайт для группы компаний MIRICO - ведущего поставщика IT-решений для автоматизации бизнеса в Узбекистане.

## Технологии

Проект построен с использованием:

- **Vite** - быстрый сборщик для фронтенда
- **TypeScript** - типизированный JavaScript
- **React** - библиотека для создания пользовательских интерфейсов
- **shadcn-ui** - коллекция переиспользуемых компонентов
- **Tailwind CSS** - utility-first CSS фреймворк
- **Framer Motion** - библиотека для анимаций
- **React Router** - маршрутизация для React приложений

## Установка и запуск

Требования:
- Node.js 18+ 
- npm или yarn

### Шаги для запуска:

```sh
# Шаг 1: Клонировать репозиторий
git clone <YOUR_GIT_URL>

# Шаг 2: Перейти в директорию проекта
cd mirico-redesign

# Шаг 3: Установить зависимости
npm install

# Шаг 4: Запустить dev-сервер
npm run dev
```

Приложение будет доступно по адресу: http://localhost:8080

## Доступные команды

```sh
npm run dev          # Запуск dev-сервера
npm run build        # Сборка для production
npm run build:dev    # Сборка в режиме разработки
npm run preview      # Предпросмотр production сборки
npm run lint         # Проверка кода линтером
npm run test         # Запуск тестов
npm run test:watch   # Запуск тестов в режиме наблюдения
```

## Структура проекта

```
mirico-redesign/
├── public/          # Статические файлы
├── src/
│   ├── components/  # React компоненты

│   ├── lib/         # Утилиты и вспомогательные функции
│   └── test/        # Тестовые файлы

```

## Разработка

При внесении изменений в код:

1. Создайте новую ветку для фичи/исправления
2. Внесите изменения
3. Запустите линтер: `npm run lint`
4. Запустите тесты: `npm run test`
5. Создайте pull request

## Деплой

Для деплоя проекта:

1. Выполните сборку: `npm run build`
2. Содержимое папки `dist/` готово к деплою на любой статический хостинг (Netlify, Vercel, GitHub Pages и т.д.)
